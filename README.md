# Birthday Animation with Turtle 🧸🎂

This is a creative birthday animation built using Python Turtle graphics.

## Features
- Custom teddy bear drawing 🧸
- Chocolate illustration 🍫
- Decorative background
- Smooth text animation (letter by letter)
- Stylish color theme

## Technologies Used
- Python
- Turtle Graphics
- Time module
- Random module

## How to Run

1. Run the program:
   python birthday.py

2. Watch the animated birthday greeting 🎉

## Output
- Displays "HAPPY BIRTHDAY" with animation
- Shows teddy bears and chocolate design
